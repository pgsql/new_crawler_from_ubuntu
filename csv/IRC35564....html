Detailed Description

SUMMARY

The Staff QA Engineer will work in a positive, friendly environment where you will have an opportunity to develop your testing skills in areas such as White Box testing.  The Staff QA Engineer will test both trading systems and market data components.  The prime directive of the position will be to ensure and certify E*TRADE’s software products meet the highest standards of quality. 

 

RESPONSIBILITIES

The position will be responsible for integration testing of order-routing systems, market data and reference data applications:

·         Create, execute, maintain and enhance test documents and testing scripts

·         Perform detailed feature and regression tests

·         Report, record, and re-verify defects

·         Responsible for retail apps automation testing
Job Requirements

REQUIREMENTS

The position requires hands-on integration testing skills on UNIX/Linux/based applications on a client-server architecture as well as Java based front end User Interface testing on Trading and Order Routing applications and Market Data systems.

·         3+ years testing experience in Unix environment

·         Sybase/Oracle working experience

·         Ability to document testing procedures, test cases and test results

·         Analytical skills to investigate and trouble-shoot of application failures, communicate issues to team leads and developers

·         Solid hands on experience of performing manual functional tests, knowledge of installation and configuration to set up testing environment

·         Good working knowledge on Software Development Life Cycle, and Integration testing methodologies and procedures

·         Ability to trace through various app logs to trouble shoot is a must

·         Excellent communication and written skills are essential

·         Basic understanding of automation methodologies

·         FIX protocol or CMS protocol or TCP/IP protocol testing experience

·         UNIX/Linux, PERL, Shell scripting

·         Financial Industry, particularly Equities, Options, and Futures trading knowledge required
