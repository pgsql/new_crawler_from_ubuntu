Detailed Description

SUMMARY

The Director, IT Architecture will lead a group responsible for developing the next generation Application and Services framework, as well as other core software components and frameworks.  As part of our strategic direction, open-source technologies are used extensively to deliver leading edge solutions.   The individual must have a proven track record in distributed, scalable, and high-performance web computing using open source technologies, and must have demonstrated successful product delivery in fast-paced environments.

 

RESPONSIBILITIES

·         Design and development (on Unix/Linux platform) of distributed web technologies and applications, service-oriented architectures (SOA including SOAP, WSDL/XDS, REST, Service Orchestration, etc.), message-oriented protocols and middleware, distributed caching, enabling software frameworks and container technologies.

·         Lead development through Agile methods and Test Driven Development.

·         Partner and collaborate with other technology leaders, stakeholders, and groups.

·         Technical/business presentations to set and define architecture and solution visions to solve challenging and practical software architecture and deployment challenges.

Job Requirements

REQUIREMENTS

·         At least 7-10 years of industry experience with a proven track record of product delivery.

·         7+ years’ experience in Java enterprise programming and Java frameworks; C/C++ and scripting languages (Perl, php, python, ruby) highly desirable.

·         7+ years’ experience developing/supporting applications on Linux/Unix platforms.

·         Solid experience with distributed web/application technologies.

·         Solid experience with open-source web and server products (Apache, Tomcat), XML and web technologies (HTML, CSS, Ajax, JavaScript)

·         Working experience with relational databases (Oracle, Sybase, SQL Server, MySQL) is highly desirable

·         Experience with development tools (Eclipse, NetBeans)

·         Experience with source code repositories (CVS, SVN) and  build tools (ANT, Maven)

·         Experience with Message Queue technologies (MOMs) and protocols

·         Strong communication and presentation skills, as well as strong/proven management skills

 

EDUCATION

·         BS or MS in Computer Science or related field
